# Project Roadmap

## Vision

Transform the QNAP QuCPE-7012 from a standalone vCPE appliance into a versatile **OpenStack edge node** capable of:

1. **L2 Gateway** - Bridging physical infrastructure to OpenStack tenant networks
2. **Edge Compute** - Running lightweight VNF workloads at remote sites
3. **Service Gateway** - Hosting distributed services close to users
